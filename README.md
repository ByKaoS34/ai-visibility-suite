# AI Visibility Suite

WordPress websites generate structured AI-readable files such as llms.txt and llms-full.txt.

## Features

- Automatic llms.txt generation
- llms-full.txt content export
- AI-friendly website information
- Category and post detection
- Page support
- Custom post type support
- Service content support
- Automatic updates when content changes

## Supported Content

- Posts
- Pages
- Custom Post Types
- Services (ourservices)

## Settings

The plugin provides settings for:

- Site name
- Description
- Phone
- WhatsApp
- Keywords
- Site type
- Main topic

## Generated Files

The plugin creates:

/llms.txt

Short AI-readable website information.


/llms-full.txt

Detailed content export for AI systems.

## Installation

1. Upload plugin folder to wp-content/plugins
2. Activate the plugin
3. Configure settings
4. llms.txt files will be generated automatically
